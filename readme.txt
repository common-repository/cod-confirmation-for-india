Using this plugin you can ensure that your COD orders are legitimate before you ship them out.
This plugin will confirm your COD orders on call or SMS as per your selected mode and you don't need to create any account on any service provider sites!

== Description ==

Cash on delivery (COD) is a great enabler for the e-commerce industry. But it also is one of the leading sources of operational loss due to cancellations and bogus orders.

Using our plugin, you can ensure that your COD orders are legitimate before you ship them out.

There are two mode is available: 

# Voice Call Mode :
- When COD order will placed than plugin will check the mode if there is voice call mode than plugin will dials the customer for confirm the order than order will confirmed base on customers input if customer will press '1' than plugin will add confirm custom field to order which will already set in admin settings for order confirm.
- There are ₹2.00 INR charge per call. 


# SMS Mode :
- When COD order will placed than plugin will check the mode if there is sms mode than plugin will sends confirmation sms to customer with order link. 
If customer open that link and press on confirm button than plugin will add order confirm tag to order in store as well plugin add custom field and take action on order on pressed button.
- There are ₹0.40 INR charge per sms. 

For SMS mode there are one order confirm page which will display order history with 'Confirm' or 'Cancel' button, on press button order will verified.

# There are two ways for check order is verified or not:

1> You can check your order custom field in your woocommerce > order detail page.

2> There are COD orders menu is available you can check all cod orders status.

# To activate this plugin & start confirming COD Orders, you have to recharge balance in your plugin using payment option (Payumoney) given in plugin, than after balance will cut on your usage bases.


Note: There are interconnections between your server and our server because order confirmation on call or SMS will be done from our server.


== Installation ==

1. Make sure woocommerce is already activate in your plugin list.
2. Upload the plugin files to the `/wp-content/plugins/COD-Confirmation` directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Use the COD Settings to configure the plugin

== Frequently asked questions ==

= What is required to confirm COD orders? =

- First you need to enable COD payment gateway on your woocommerce payment section.
- Ensure that you have enough balance in your COD plugin if not than recharge your balance though payumoney on press 'Recharge now' button.
- Check your COD plugin mode (Voice call or SMS).

= How can I check order is confirm by SMS or Voice call? =

- There are COD orders listing is available with your usage balance, verified by and order status (cancel,confirm or pending).
